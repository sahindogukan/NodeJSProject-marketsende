# marketsende

**marketsende** piyasadaki market ürünlerinin güncel indirimlerini kullanýcýlarýna sunmayý amaçlayan NodeJS tabanlý bir web projedir. Demo giriþi ile admin panelinden birçok ürün ve arayüz ayarlarý dinamik þekilde güncellenebilir.
